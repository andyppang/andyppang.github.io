组图

如果想把多张图片按一定布局组合显示，你可以在 markdown 中按如下格式：

total：图片总数量，对应中间包含的图片 url 数量 n1-n2-...：每行的图片数量，可以省略，默认单行最多 3 张图，求和必须相等于 total，否则按默认样式

例如 {% gi 5 3-2 %} 代表共 5 张图，第一行 3 张图，第二行 2 张图。
